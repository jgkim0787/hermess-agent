AI Agent 입문 — Hermes 과정

개념을 이해하고(설명), 직접 써보고(실습), 나만의 미니 에이전트 Hermes를 한 줄씩 만들며(복제) AI Agent를 익힙니다. Python + Claude API.

학습 로드맵

이 과정의 사용법

  • 각 모듈은 개념 → 직접 사용해보기 → Hermes 만들기 3단계로 구성됩니다.
  • 학습 중 궁금한 점을 물으면 답변과 함께 질문 모음에 자동 기록됩니다.
  • 중간/완료 시 그동안의 질문으로 복습(리마인드) 자료를 만들 수 있습니다.
  • 복제 프로젝트 코드는 저장소의 clone-project/ 폴더에 직접 타이핑하며 쌓아갑니다.
다음에 학습하려면 — Claude Code 세션에서 "다음 강의 배우자" 또는 "1강 만들어줘" 라고 하면 학습 하네스가 해당 모듈 자료를 생성합니다.
